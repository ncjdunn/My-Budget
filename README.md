# Autopay Budget PWA

A React + TailwindCSS Progressive Web App to manage autopay bills, visualize due dates, and summarize weekly/monthly budgets.

## Setup

1. **Install dependencies**
   ```bash
   npm install
   ```
2. **Start development server**
   ```bash
   npm start
   ```
3. **Build for production**
   ```bash
   npm run build
   ```
